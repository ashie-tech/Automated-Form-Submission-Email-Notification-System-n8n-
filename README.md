# 🚀 n8n Automated Form Processing Workflow

## 📌 Overview

This project demonstrates an automated workflow built using n8n.
The workflow captures form submissions, stores them in Google Sheets, and sends an email notification via Gmail.

## 🛠 Tools Used

1. n8n

2. On form Subimmision Trigger

3. Google Sheets Node

4. Gmail Node

## ⚙️ Workflow Logic
Webhook receives form data

Data is processed and appended to Google Sheets

Automated email notification is sent via Gmail
